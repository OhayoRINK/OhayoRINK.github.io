Lightweight Baseline

PekoraNet 🐇

A Lightweight Hierarchical CNN Framework for CSI Feedback Compression

Motivation

PekoraNet focuses on practical deployment feasibility for UE-side CSI compression by combining hierarchical CNN encoding and lightweight reconstruction refinement.

Key Features

  • Hierarchical CNN encoder
  • Residual reconstruction refinement
  • Low-parameter lightweight design
  • Stable low-CR reconstruction

Architecture

Conv2d(2→16) ↓ Conv2d(16→32) ↓ Latent Compression ↓ ConvTranspose2d ↓ Residual Reconstruction
Indoor 1/4 NMSE -22.067 dB
Parameters ~2.1M
Spectral Learning

DPAS-CsiNet

Dual-Path Adaptive Spectral CSI Compression Architecture

Motivation

DPAS-CsiNet extends the original CsiNet by introducing spectral-domain processing and adaptive frequency refinement mechanisms.

Key Features

  • Dual-Path encoder
  • Adaptive Frequency Gating (AFG)
  • Spectral Refinement Block (SRB)
  • FFT-domain channel refinement

Architecture

Input CSI ↓ Dual-Path Encoder ↓ Adaptive Frequency Gating ↓ Decoder ↓ Spectral Refinement Block
Indoor 1/4 NMSE -24.72 dB
Core Feature FFT Learning
Latest Architecture

TowaNet 👾

Topology-Oriented Wireless Autoencoder for CSI Compression

Motivation

TowaNet combines the spectral learning strategy of DPAS with the hierarchical compression philosophy of PekoraNet to achieve stronger reconstruction performance and AMP-safe refinement.

Key Features

  • Stride=2 spatial compression
  • TowaRefineBlock
  • Decoder-centric refinement
  • AMP-compatible architecture

Architecture

Input CSI ↓ Spatial Compression Encoder ↓ Latent Codeword ↓ 16×16 Latent Reconstruction ↓ TowaRefineBlock ×2 ↓ Recovered CSI
Indoor 1/4 NMSE -26.50 dB
Architecture AMP-safe
